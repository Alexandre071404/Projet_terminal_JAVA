All the games contained in the project are played in the terminal.

Our packages are organized as follows:

First Part:
We have three files in each package:

One file contains the class named "laclasse.java."

Then we have the executable file, which is named "Demo.java" in the "nim" package and "main.java" in the "tictactoe" package.

Finally, we have the tests contained in the "Test.java" file.

Second Part:
We have four directories containing various classes, each with its respective tests.

To execute the final result of our project, you need to compile the entire "games" directory.
Then launch the "games.plays.Main" file.
